Welcome.  This zip file contains a build for the VR version of this application.

To extract the build download all files to the samle folder/location: VR_v4.zip, VR_v4.z01 and VR_v4.z02.  Then extract the zip file to your desired location.  

These files have been partitioned to meet the commit limit of github. Failing to extract with all three files will result in a corrupted application.
